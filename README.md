# backend-coding-challenge
This microservice allows you to get the information about most used languages on github trending repositories, This is a solution to a problem given by Gemography, Lumen framework is used in order to answer the needs. Lumen is a solution to build laravel microservices and blazing fast APIS. It is classified as one of the fastest micro frameworks available.

# Connect to the server from local
In order to connect to the server using lumen framework, it is favorable to use vagrant homestead or with the following command : php -S localhost:8000 -t public

# Root used 
```php
$router->get('/github','languagesController@getLanguages');
```
This root is used to return the requested data.
